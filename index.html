1. Tipos de dados coletados

Este aplicativo foi desenvolvido com o objetivo de oferecer uma experiência de uso segura e respeitosa à privacidade do usuário. O aplicativo não coleta dados pessoais, como endereço de e-mail, foto de perfil do Google, nome e sobrenome ou quaisquer outras informações que possam identificar o usuário individualmente.

O foco principal do aplicativo é oferecer funcionalidades úteis de forma eficiente, sem comprometer a privacidade do usuário. O aplicativo não utiliza cookies ou outras ferramentas de rastreamento para coletar informações sobre o uso do aplicativo.

A equipe de desenvolvimento está comprometida em garantir que o aplicativo seja utilizado de forma segura e transparente, sem a necessidade de coleta de dados pessoais, proporcionando uma experiência de uso livre de preocupações com a privacidade.

2. Modo e local de processamento dos dados

Método de processamento

Este aplicativo foi desenvolvido com o intuito de oferecer uma experiência de uso independente de acesso à internet, garantindo a privacidade e segurança dos dados do usuário. O processamento de informações inseridas no aplicativo ocorre integralmente no dispositivo móvel do usuário, sem necessidade de conexão com a rede. Essa característica garante que os dados permaneçam confidenciais e protegidos, sem serem transmitidos para servidores externos ou compartilhados com terceiros. A funcionalidade completa do aplicativo é realizada através de recursos locais do dispositivo, assegurando a privacidade e a segurança dos dados do usuário.

Lugar

Os dados são processados localmente no dispositivo, apenas.

Período de conservação

O aplicativo, ao ser instalado no dispositivo móvel, armazena apenas os dados essenciais para a sua funcionalidade, os quais são mantidos no dispositivo até que o aplicativo seja desinstalado. A retenção desses dados é estritamente limitada ao período de utilização do aplicativo, sendo excluídos de forma permanente no momento da desinstalação, garantindo que nenhuma informação do usuário seja mantida no dispositivo após a remoção do aplicativo. Essa prática assegura a privacidade e o controle do usuário sobre os seus próprios dados, permitindo que o aplicativo seja utilizado de forma segura e transparente.

O Uso dos dados coletados

Este aplicativo não coleta ou armazena nenhum tipo de log ou relatório de uso do usuário. A experiência do usuário é completamente privada, sem qualquer registro de atividades ou dados de desempenho. O aplicativo não utiliza informações pessoais para fins de estatísticas ou análises de desempenho.

3. Informações detalhadas sobre o processamento de dados pessoais

Este aplicativo foi concebido para operar completamente offline, realizando todas as operações de processamento de dados diretamente no dispositivo do usuário, sem qualquer necessidade de interação com a internet.

Essa característica garante a segurança e a privacidade dos dados do usuário, pois as informações não são transferidas para servidores externos ou compartilhadas com terceiros. Em virtude dessa característica fundamental, o aplicativo não mantém registros detalhados sobre o processamento de dados pessoais, uma vez que todas as operações são executadas localmente.

Os relatórios gerados pelo aplicativo, disponibilizados em formato PDF, são destinados exclusivamente ao uso individual do usuário, não sendo armazenados ou transmitidos para servidores externos. Essa abordagem garante a confidencialidade e a proteção dos dados, sem a necessidade de compartilhamento ou coleta centralizada.

A ausência de coleta ou armazenamento de dados detalhados sobre o processamento de informações pessoais reforça o compromisso do aplicativo com a privacidade e a segurança dos dados do usuário, proporcionando uma experiência de uso livre de preocupações com a exposição ou o compartilhamento de informações confidenciais.

4. Informações adicionais sobre a coleta e processamento de dados

Ação jurídica

A ausência de coleta ou compartilhamento de dados pessoais garante a segurança e a privacidade do usuário, isentando o desenvolvedor de quaisquer responsabilidades legais relacionadas ao tratamento de dados pessoais. O usuário pode ter certeza de que o aplicativo não coleta ou utiliza os seus dados pessoais para fins comerciais ou outros, garantindo uma experiência de uso segura e transparente.

5. Informações adicionais sobre os dados pessoais do usuário

Além das informações contidas nesta política de privacidade, este aplicativo poderá fornecer ao usuário informações adicionais e contextuais sobre os serviços específicos ou a coleta e processamento de dados pessoais mediante solicitação.

6. Logs do sistema e manutenção

Para fins de desenvolvimento e manutenção, o aplicativo utiliza os serviços da Google para coletar logs de uso anônimos, que não identificam o usuário individualmente. Essa coleta de dados é estritamente limitada a informações técnicas e de uso, visando melhorar a funcionalidade e a estabilidade do aplicativo. Os dados coletados não incluem informações pessoais do usuário, e são utilizados exclusivamente para fins de aprimoramento do aplicativo, sem qualquer finalidade comercial ou de compartilhamento com terceiros.

7. As informações não contidas nesta política

Mais detalhes sobre a coleta ou processamento de dados pessoais podem ser solicitados ao controlador de dados, a qualquer momento. Favor ver as informações de contato no início deste documento.

8. Os direitos dos usuários

Os usuários têm o direito de, a qualquer tempo, consultar o controlador de dados para saber se os seus dados pessoais foram armazenados e saber mais sobre o conteúdo e origem, verificar a sua exatidão ou para pedir que sejam complementados, cancelados, atualizados ou corrigidos, ou que sejam transformados em formato anônimo ou bloquear quaisquer dados mantidos em violação da lei, bem como se opor ao seu tratamento por quaisquer todas as razões legítimas. Os pedidos devem ser enviados para o controlador de dados usando a informação de contato fornecida anteriormente.

Este aplicativo não suporta pedidos de “Não me rastreie”.

Para determinar se qualquer um dos serviços de terceiros que utiliza honram solicitações de “Não me rastreie”, por favor leia as políticas de privacidade.

9. Mudanças nesta política de privacidade

O controlador de dados se reserva o direito de fazer alterações nesta política de privacidade a qualquer momento, mediante comunicação aos seus usuários nesta página. É altamente recomendável que esta página seja consultada várias vezes em relação à última modificação descrita na parte inferior. Se o Usuário não concorda com qualquer das alterações da política de privacidade, o usuário deve cessar o uso deste aplicativo e pode solicitar ao controlador de dados que apague os dados pessoais dele. Salvo disposição em contrário, a atual política de privacidade se aplica a todos os dados pessoais dos usuários que o controlador de dados tiver.

Dados pessoais

Quaisquer informações relativas a uma pessoa física, pessoa jurídica, instituição ou associação, as quais sejam, ou possam ser identificadas, mesmo que indiretamente, por referência a quaisquer outras informações, incluindo um número de identificação pessoal.

Dados de uso

As informações coletadas automaticamente a partir deste serviço (ou serviços de terceiros contratados neste aplicativo, que podem incluir: os endereços IP ou nomes de domínio dos computadores utilizados pelos usuários que utilizam este aplicativo, os endereços URI (Identificador Uniforme de Recurso), a a e hora do pedido, o método utilizado para submeter o pedido ao servidor, o tamanho do arquivo recebido em resposta, o código numérico que indica o status do servidor de resposta (resultado positivo, erro , etc.), o país de origem, as características do navegador e do sistema operacional utilizado pelo usuário, os vários detalhes de tempo por visita (por exemplo, o tempo gasto em cada página dentro do aplicativo) e os detalhes sobre o caminho seguido dentro da aplicação, com especial referência à sequência de páginas visitadas e outros parâmetros sobre o sistema operacional do dispositivo e/ou ambiente de TI do Usuário.

Usuário

A pessoa que usa este aplicativo que deverá coincidir com ou estar autorizada pelo titular dos dados a quem os dados pessoais se referem.

Titular dos dados

Controlador de dados (ou proprietário)

O controlador de dados, a menos que seja especificado de outra forma, é o proprietário deste serviço (este aplicativo).
